API REST desenvolvida em **Java com Spring Boot**

Implementa um **CRUD de usuários** com:
- criação
- listagem
- busca por ID
- desativação lógica
- exclusão

## Tecnologias
- Java 17
- Spring Boot
- Spring Data JPA
- Jakarta Validation
- H2 Database
- Maven

## Endpoints
- `POST /users`
- `GET /users`
- `GET /users/{id}`
- `PUT /users/{id}/desactivate`
- `DELETE /users/{id}`
